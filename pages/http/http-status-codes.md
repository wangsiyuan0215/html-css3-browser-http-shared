# HTTP 状态码

HTTP 响应状态码用来表明特定 HTTP 请求是否成功完成。 响应被归为以下五大类：

- 信息响应 **100–199**
- 成功响应 **200–299**
- 重定向消息 **300–399**
- 客户端错误响应 **400–499**
- 服务端错误响应 **500–599**
